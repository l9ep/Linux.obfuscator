Firestore Security Rules
Paste these rules in Firebase Console → Firestore → Rules:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /scripts/{scriptId} { // Anyone can read scripts (for loadstring) allow read: true; // Only authenticated owner can write/delete allow write, delete: if request.auth != null && request.auth.token.email == "YOUR_EMAIL@gmail.com"; } match /stats/{doc} { allow read: true; allow write: if true; // load counter } } }
